# CodeAthlon Backend 🚀

A backend system for a coding interview preparation platform that provides structured daily workouts, tracks user progress, and maintains streaks.

## 🔥 Features

- 🎯 **Workout API**
  - Generates daily coding problems based on company (Amazon, Google, etc.)
  - Balanced difficulty (Easy + Medium)

- 📈 **Progress API**
  - Tracks topic-wise progress (Arrays, Strings, DP, etc.)
  - Returns percentage completion

- 🔥 **Streak API**
  - Tracks daily solving streak
  - Encourages consistency

## 🛠️ Tech Stack

- Java
- Spring Boot
- REST APIs

## 📌 API Endpoints

- `/workout?company=Amazon`
- `/streak`
- `/progress`
